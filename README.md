#cl-opencc

cl-opencc is a Common Lisp bindings for [opencc](https://github.com/BYVoid/OpenCC)

## Install

### Ubuntu 17.10 and Roswell

```
apt install libopencc-dev
```


Install cl-opencc

```
ros install libgirlenterprise/cl-opencc
```

## Test

```lisp
(asdf:test-system :cl-opencc)
```
